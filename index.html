Ying 自動化教學 - Robot Framework

Robot Framework 是一個不需要有任何程式語言的工具。讓沒有程式語言背景的測試工程師跳過艱澀的程式語言邏輯也可以順利的讓測試自動跑起來。

目標教學 – 建立自動化教學的初衷，有許多人想要從手動測試工程師轉為自動測試工程師。卻因為英文的門檻而無法順利地學習。希望可以藉由這個網站降低語言的門檻並且幫助想要擁有多項技能的測試工程師成長。

來～來～接下來讓我講講Robot Framework的運用邏輯是什麼吧～ 你可以把它想像成... 一個指令一個動作的操作邏輯。假設我們要用瀏覽器打開一個網頁。這個時候我就會告訴電腦，使用關鍵字 “Open Browser” 打開網址 https://www.yahoo.com.tw 而且指定使用 chrome 瀏覽器開啟。最後再關掉瀏覽器。

這時候完整的腳本就是像下面的樣子：

  • Open Browser https://www.yahoo.com.tw chrome
  • Close Browser
